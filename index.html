Pega Pega Sonic x Robotnik

Este é um jogo de perseguição divertido e educativo desenvolvido na plataforma SCRATCH. Nele, os jogadores controlam o Sonic em uma emocionante corrida para capturar o Robotnik, enquanto desviam de obstáculos e coletam itens. Ideal para introduzir conceitos de programação e lógica de forma lúdica.

Lançamento de Foguetes

Este projeto incrível foi desenvolvido em colaboração com meus alunos. Utilizamos garrafas PET e ar comprimido com água para criar e lançar foguetes caseiros. Uma atividade prática que explora princípios de física como propulsão, aerodinâmica e pressão, estimulando o interesse científico e o trabalho em equipe.

Ping Pong Multiplayer

Um clássico jogo de Ping Pong recriado no P5.js, permitindo que dois jogadores compitam entre si. Este projeto demonstra conceitos de programação como detecção de colisão, controle de movimento e lógica de jogo. É uma ótima ferramenta para ensinar interatividade e programação criativa.
